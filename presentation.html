Project Goal

專案目標

♿

無障礙導航

提供校園內 AED、SOS 緊急電話、無障礙坡道與電梯的即時位置資訊

🌍

SDGs 對齊

呼應聯合國永續發展目標，促進包容性與公平性，讓每位師生都能安全移動

📱

行動友善

響應式設計，支援手機瀏覽器地理定位，快速找到最近無障礙設施

Features · Done

已完成功能

  • ✓
    互動式地圖：Leaflet 地圖，支援街道與衛星雙圖層切換，限定中央大學校園範圍
  • ✓
    設施標記：AED、SOS 緊急電話、無障礙坡道、無障礙電梯，共 25+ 個地點
  • ✓
    用戶定位：GPS 定位、精度顯示（±公尺）、脈動藍點標示、最近設施推薦
  • ✓
    即時搜尋：模糊搜尋設施名稱與描述，最多顯示 8 筆結果
  • ✓
    側邊欄篩選：各類別設施顯示/隱藏切換，附設施數量徽章
  • ✓
    問題回報 Modal：Formspree 整合表單，支援位置錯誤、損壞、遺漏等回報類型
  • ✓
    使用說明 Modal：操作指引、地圖說明、圖例介紹
  • ✓
    Toast 通知：定位成功/失敗、操作回饋，3.2 秒後自動消失
